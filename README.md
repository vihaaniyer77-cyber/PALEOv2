## 📖 Overview

**PALEO** is an advanced machine-learning pipeline purpose-built for the classification of exoplanet transits and stellar flares from photometric light curve data (e.g., TESS). At its core, PALEO leverages a powerful **Temporal Convolutional Network (TCN)** to efficiently and accurately identify astronomical anomalies, automating the time-consuming process of light curve analysis. 

With its ability to distinguish between planetary transits and stellar flares, PALEO is designed to accelerate exoplanet discovery and support impactful observational research proposals.

## ✨ Key Features

- **🧠 TCN-Based Classification:** Utilizes a custom Temporal Convolutional Network (currently operating on `tcn_epoch_20.pt`) specifically trained on light curve sequences.
- **🌟 Multi-Class Detection:** Robust, standalone classification tailored to distinctly identify **Exoplanet Transits** and **Stellar Flares**, minimizing false positives.
- **📊 Automated Visualization:** Built-in capabilities for generating standalone plots and periodograms for classified transits and flares directly from pipeline results.
- **📈 Dataflow Optimization:** Designed to handle complex light curve extraction, filtering, cleaning, and model-inferencing at scale.
