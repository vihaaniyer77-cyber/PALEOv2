## 📖 Overview

**PALEO** is an advanced machine-learning pipeline purpose-built for the classification of exoplanet transits and stellar flares from photometric light curve data (e.g., TESS, Kepler). At its core, PALEO bridges deep learning and physical modeling by combining a powerful **Temporal Convolutional Network (TCN)** with a robust **Physics-Aware Validator** to efficiently and accurately identify astronomical anomalies.

With its ability to distinguish between planetary transits, stellar flares, and eclipsing binaries (EBs) using physically motivated constraints, PALEO is designed to accelerate exoplanet discovery and support impactful observational research proposals.

## ✨ Key Features

- **🧠 TCN-Based Classification:** Utilizes a custom Temporal Convolutional Network specifically trained on light curve sequences for fast, high-confidence anomaly anomaly detection.
- **🔭 Physics-Aware Validator:** An integrated analytical verification layer that evaluates candidates using physical constraints (e.g., transit depth, duration, and "Physics Gain"). This acts as a critical post-processing filter to screen out false positives like EBs and complex noise.
- **🌟 Multi-Class & Flare-Aware:** Robust classification designed to cleanly isolate Exoplanet Transits from Stellar Flares, featuring flare-aware dataset splitting to ensure rigorous out-of-sample evaluation.
- **📊 Automated Visualization:** Built-in capabilities for generating transit and flare classification visualizations.
- **📈 Dataflow Optimization:** Engineered for the efficient extraction, windowing, and caching of raw FITS light curves at scale.

## 💻 Getting Started

### Prerequisites

Ensure you have Python 3.8+ installed along with the required machine learning and astronomical packages:

```bash
pip install torch numpy pandas matplotlib lightkurve astropy
```

### Usage

1. **Load your Model:** Ensure your pre-trained weights (e.g., `tcn_epoch_20.pt`) are in your working directory.
2. **Run Inference & Validation:** Process your light curve data through the TCN and the subsequent Physics-Aware Validator to score anomalies and rank them by Physics Gain.
3. **Review Results:** Use the built-in plotting utilities to export transit and flare classification visualizations.

*(Detailed documentation and code examples on utilizing both the standalone TCN model and the full physics-aware pipeline are available in the Jupyter notebooks included in this repository).*
